# Automated Optical Inspection (AOI) System Prototype
本项目旨在设计、制造和编程一款桌面式自动光学检测 (AOI) 系统原型，以模拟工业半导体计量流程。该系统将硬件工程与计算机视觉相结合，利用 Python 和 OpenCV 实现的“黄金样本”对比方法，识别印刷电路板 (PCB) 上的表面缺陷。项目重点在于构建稳定的光学装置以实现一致的成像效果，开发高精度异常检测算法，并建立数据管道将检测结果记录到 SQL 数据库中。因此，该项目模拟了大批量生产的工作流程，并利用 Tableau 进行良率分析，以展示精密工程和数据分析在半导体领域的实际应用。

—

This project involves the design, fabrication, and programming of a desktop Automated Optical Inspection (AOI) system prototype aimed at simulating industrial semiconductor metrology processes. By integrating hardware engineering with computer vision, the system identifies surface defects on printed circuit boards (PCBs) using a "Golden Sample" comparison method implemented in Python and OpenCV. The initiative focuses on constructing a stable optical rig for consistent imaging, developing algorithms to detect anomalies with high accuracy, and establishing a data pipeline to log inspection results into an SQL database. Thus, the work replicates high-volume manufacturing workflows, utilizing Tableau for yield analysis to demonstrate the practical application of precision engineering and data analytics within the semiconductor domain.
